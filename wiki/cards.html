Main Boss at Mini Boss Cards – Nagtatala ang mga ito ng lakas at kakayahan ng mga kalabang haharapin ninyo.

Item Cards

Special Item Cards – Nakukuha ang mga ito mula sa mga natalong mini boss at nagagamit upang palakasin o protektahan ang inyong grupo sa mga susunod na Dungeon.

  • Balaraw – ×2 Dungeon Buff (+5% → +15% DMG to all).
    Makukuha pagkatapos talunin ang dungeon mini boss ng Dagat ng Kabisayaan.
  • Kalasag – ×1 Dungeon Buff (+15% → +25% DEF to 2 players).
    Makukuha pagkatapos talunin ang mini boss ng Daragang Magayon.
  • Agos-Oras – ×4 rounds no CD (lahat ng skills walang cooldown sa susunod na round).
    Makukuha pagkatapos talunin ang mini boss ng Bundok Pulag.

Turn Pieces

Mga palatandaan na ginagamit upang masubaybayan ang cooldown ng bawat skill ninyo.

Playing Mat

Ang lugar kung saan ipupwesto ng mga player ang kanilang mga cards na napili.

4) Introduction to Dice

  • d4 – tumutukoy kung sino ang tatargetin ng halimaw o diyos.
  • d10 – tumutukoy sa halaga ng epekto o pinsala ng skill o espesyal na galaw.
  • d20 – para sa bawat atake. (1 = mintis, 20 = critical, ang iba ay normal).
  • d100 – para sa kakaibang pangyayari o espesyal na kaganapan—dito sinusubok ang swerte o malas.

5) Core Loop

  1. Choose Role → Select Preset Stats → Pick 3 Skills.
  2. Voyage → Choose Map → Encounter Mini Boss → Optional Mini Boss Fight.
  3. Victory → Gain Special Drops (Item Cards - Boost Stats).
  4. Continue the Dungeon → Encounter Main Boss → Main Boss Fight.
  5. Victory → Gain Special Drops (Memory Fragment - Relic).
  6. Return to Balangay Hub → Plan ahead and choose the next dungeon.
  7. Next Voyage → Repeat until all 4 Main Bosses are defeated.
  8. All 4 Main Bosses Defeated → Build the 4 Memory Fragments → Victory → End of Game.
